# Gustavo in the Jungle

A top-down action RPG where Gustavo, a short bespectacled round fellow in cargo shorts, battles jungle creatures to earn XP and level up. No story, no quests -- pure combat and progression built with pygame.

## Install

```
pip install -r requirements.txt
```

## Run

```
cd gustavo_jungle && python main.py
```

## Controls

- **WASD** -- Move
- **Left-click** -- Melee attack
- **1-4** -- Skills
- **ESC** -- Pause
